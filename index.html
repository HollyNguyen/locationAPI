Location API

Overview

This amazing location API allows access to a dataset of locations (CSV format) with the following informaiton included. Response return IDs which can be matched to the location and other information about that specific row of data.

Dataset Information

Data features:
  • ID
  • Location name
  • State abbreviation
  • Elevation
  • Type of location
Example data:
  • ID: 205707
  • Location name: Breakneck Pond
  • State abbreviation: CT
  • Elevation: 696
  • Type of location: Lake
