Location API

Overview

This amazing location API allows access to a dataset of locations (JSON format) with the following information included. The response returns IDs which can be matched to the location and additional information about that location.

Dataset Information Included

Data features:

  • ID
  • Location name
  • State abbreviation (must be the abbreviation)
  • Elevation (in meters)

Example data:

  • ID: 205707
  • Location name: Breakneck Pond
  • State abbreviation: CT
  • Elevation: 696

Example use:

This is how you search for a location based on state

/search?STATE_ALPHA=MA

This is how you search for a range of elevations

/search?elevationmin=200

You can combine search terms (search for a state and a minimum elevation

/search?STATE_ALPHA=MA&elevationmin=200

This returns any locations in Massachusetts with an elevation greater than 700m
